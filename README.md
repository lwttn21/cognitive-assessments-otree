# Mental Fatigue & Cognitive Tasks Repository

This repository provides a collection of oTree tasks designed to **generate** and **measure** mental fatigue.

The entire collection is implemented as a single **oTree 5 project**. Each task functions as a standalone app, making it easy to integrate specific components into your own research.

## How to Use

* **Demo:** You can run this project to view and test every individual task.
* **Integration:** To use a task in your own experiment, simply copy the corresponding app folder into your project and adjust the code as needed.
